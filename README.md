
# ❤️ Heart Disease Prediction App

A web-based AI application that predicts the likelihood of heart disease based on user health inputs. This tool helps users and healthcare professionals make early risk assessments using machine learning.

🌐 Live App: https://freeheartdiseaseprediction.netlify.app/

---

## 🚀 Features

- 🧠 AI-based heart disease prediction
- 📊 User-friendly input form
- ⚡ Instant prediction results
- 🌍 Accessible from anywhere (web-based)
- 💡 Simple and clean UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / Streamlit (if used)
- Machine Learning: Python (Scikit-learn / Pandas / NumPy)
- Deployment: Netlify

---

## 📥 Input Parameters

The model takes various health-related inputs such as:

- Age
- Gender
- Chest Pain Type
- Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise-Induced Angina

---

## 📤 Output

- Prediction: **Heart Disease Risk (Yes / No)**
- Based on trained ML model

---

## 🧠 How It Works

1. User enters medical data
2. Data is processed and sent to the ML model
3. Model predicts risk level
4. Result is displayed instantly
